Summary: Latitude vs. X

The purpose of this project was to see if there is a relationship between latitude and four different weather events: max temperature, wind speed, humidity, and cloud cover. Five hundred random cities were selected on March 15, 2020 to then use a weather API to observe the cities' weather on that day and how it related to the four weather factors.

While this is a representative model of weather in cities across the world, it is only a snapshot of one day's weather - March 15, 2020. A seasonal or yearly historical view could offer more insight. This site contains the visualizations, analysis, and source data.
